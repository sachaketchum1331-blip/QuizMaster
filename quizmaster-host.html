🔧 Configuration Firebase

À faire une seule fois — 5 minutes pour une base de données temps réel gratuite, hébergée en Europe.

📋 Guide pas à pas

1

Allez sur console.firebase.google.com → Créer un projet (nom au choix) → désactivez Google Analytics

2

Menu gauche → Realtime Database → Créer une base de données → région europe-west1 (Belgique) → mode test

3

⚙️ Paramètres du projet → Général → Vos applications → icône </> → Enregistrez l'app web

4

Dans la config affichée, copiez la valeur de databaseURL (ex: https://mon-projet-rtdb.europe-west1.firebasedatabase.app) et collez-la ci-dessous

URL Realtime Database
✅ Connexion réussie !
